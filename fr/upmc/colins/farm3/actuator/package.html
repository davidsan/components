Actuator

This package contains the actuator component and his dynamic version. This package also contains all classes affiliated with the actuator.

An actuator is a component that will forward received requests to its dedicated virtual machine.

A target service time is set. The mean service time sent by the request dispatcher is processed and an action is taken if the mean service time is too slow or if it is too fast. A flex time is added to make the target service time for flexible.
